# David's Config

Here are my config files!

## Instructions

Clone the repository. Then, install the config.

```
$ git clone git@github.com:davidfcohen/config.git
$ cd config
$ ./install.sh
```

Optionally, create symbolic links manually.

```
$ ln -sn $(pwd)/alacritty $HOME/.config/alacritty
```
