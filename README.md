# David's Dotfiles

## Instructions

Clone the repository and create symbolic links.

```
$ git clone git@github.com:davidfcohen/dotfiles.git
$ cd dotfiles
$ ln -sr alacritty/ ~/.config/alacritty
```

| Dotfile(s)     | Location              |
| -------------- | --------------------- |
| `alacritty`    | `~/.config/alacritty` |
| `fish`         | `~/.config/fish`      |
| `helix`        | `~/.config/helix`     |
