# David's Dotfiles

Clone this repository and link the desired files to their correct location.

```
$ git clone git@github.com:davidfcohen/dotfiles.git
$ cd dotfiles
$ rm -i ~/.bashrc
$ ln -s ./bashrc ~/.bashrc
```

| File           | Program | Location          |
| -------------- | ------- | ----------------- |
| `bashrc`       | Bash    | `~/.bashrc`       | 
| `bash_profile` | Bash    | `~/.bash_profile` |
| `helix`        | Helix   | `~/.config/helix` |
